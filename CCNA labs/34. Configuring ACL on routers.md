## Goal

1.  Configure OSPF on R1 and R2 allow full connectivity between the PCs and servers . 
2. Configure the standard number ACLs on R1 and standard named ACLSs on R3 to full fill the following network policies.
    a. Only PC1 and PC3 can access 192.168.1.0/24
    b. Hosts in 172.16.2.0/24 can't access 192.168.2.0/24
    c. 172.16.1.0/24 can't access 172.16.2.0/24
    d. 172.16.2.0/24 can't access 172.16.1.0/24

## Configuration - OSPF (on R1/R2)

```
R1(config)interface range g0/0,g0/1
R1(config)ip ospf 1 area 0
R1(config)interface range s0/0/0
R1(config)ip ospf 1 area 0
# configure same as g0/0,g0/1,s0/0/0 on R2
R1(config)do sh ip rout
#check out
```


## Configuration - ACL (R2)

 - requirement only PC1 and PC3 can access 192.168.1.0/24

```
R2(config)access-list 1 perimt 172.16.1.1
R2(config)access-list 1 perimt 172.16.2.1
R2(config)access-list 1 perimt any
R2(config)interface g0/0
R2(config-if)ip access-group 1 out
R2(config-if)do sh access-list
```

## Configuration - ACL (R2)

- requirement : hosts in 172.16.2.0/24 can't access 192.168.2.0/24

```
R2(config)access-list 2 deny 172.16.2.0 0.0.0.255
R2(config)access-list 2 permit any
R2(config)interface g0/1
R2(config-if)ip access-group 2 out
R2(config-if)do sh access-list
```

##  Configuration - ACL (R1)

 requirement :
 
 172.16.1.0/24 can't access 172.16.2.0/24
 172.16.2.0/24 can't access 172.16.1.0/24

```
R1(config)ip access-list standard deny-172.16.1.0
R1(config-std-nacl)deny 172.16.1.0 0.0.0.255 
R1(config)ip access-list standard deny-172.16.2.0
R1(config-std-nacl)deny 172.16.2.0 0.0.0.255 
R1(config)interface g0/1
R1(config-if)ip access-group deny-172.16.1.0 out
R1(config)interface g0/0
R1(config-if)ip access-group deny-172.16.2.0 out
