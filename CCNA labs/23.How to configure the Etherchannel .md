## Date : 2025-12-16

## Goal

1. Understand how to verify and configure the Etherchannel.
2. Configure Distribution Switches (DSW1/DSW2) to perform Layer 3 routing so that PCs can reach the server.
3. Verify and understand load-balancing behavior on switches.

 ## **Configuration**(LACP)
 ```
 ASW1(config) interface (range) xxx 
 ASW1(config-if-range)channel-group 1 mode active
 ASW1(config-if-range)interface prot-channel 1 
 ASW1(config-if)switchport mode trunk
 ASW1(config-if)do show etherchannel summary
 ASW1(config-if)do show interface trunk
 ASW1(config-ife)do show spanning-tree
 ```

 ## **Configuration**(PAgP)[Layer 2]
 ```
 ASW1(config) interface (range) xxx 
 ASW1(config-if-range)channel-group 1 mode desiable
 ASW1(config-if-range)interface prot-channel 1 
 ASW1(config-ife)switchport mode trunk
 ASW1(config-if)do show etherchannel summary
 ASW1(config-if)do show interface trunk
 ```

## **Configuration**(static etherchannel)[Layer 3]
 ```
 DSW1(config) interface (range) xxx 
 DSW1(config-if-range)no switchport
 DSW1(config-if-range)channel-group x mode on
 DSW1(config) interface pox  
 
 # x means the number of channel group 

 DSW1(config-if)ip address x.x.x.x 255.255.255.252
 DSW1(config-if)do show etherchannel summary
 DSW1(config-if)do show interface trunk
 ```
