## Configuration (on R1/2/3/4) - OSPF

```
R1(configure)router ospf 100     #100= number of ospf in the local router
R1(configure-router) 10.0.13.0 0.0.0.3 area 0
R1(configure-router) 10.0.34.0 0.0.0.3 area 0
R1(configure-router) 1.1.1.1  0.0.0.0 area 0

 # network IP wildmask area 0 /actually there are several way to set the network , which inculde 0.0.0.0 255.255.255.255 (DOnot command in real situation)
```
```
R1(configure-router) passive-interface loopback 0

# configure the passive interface , which inculde the loopback interface
```


## Configuration (on R1/2/3/4) - advertise default routes

```
R1(configure)router ospf 100
R1(configure-router)dafault-inforamtion originate
R1(configure)ip route 0.0.0.0 0.0.0.0 203.0.113.2 
```
