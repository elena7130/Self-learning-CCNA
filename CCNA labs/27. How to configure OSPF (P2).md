## Configuration (on R1/2/3/4) - OSPF )(configure directly on interface )

```
R1(config)interface range g0/0 , f1/0 , l0
R1(config-if-range)ip ospf 1 area 0
R1(config-if-range)router ospf 1
```


## Configuration (on R1/2/3/4) - change cost on each OSPF

